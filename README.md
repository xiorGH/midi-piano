# English
This is a piano I've made using [**soundfont-player** by *DaniGB*](https://github.com/danigb/soundfont-player) (which is based on [**midi-js-soundfonts** by *Gleitz*](https://github.com/gleitz/midi-js-soundfonts)) as a sound manager for this piano. As I mentioned, this project was made for a request made by a partner (in consecuence, it becomed a method to get extra points) but I've excited a little bit and added more things than I should (but hey, at least it makes it looks more professional, even if the code isn't 100% polished or made entirely by me)
